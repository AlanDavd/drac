# Drac

Drac interpreter for a C-like programming language.

## Features

- C-like syntax
- Variable bindings
- Integers and booleans
- Arithmetic expressions
- Built-in functions
- First-class and higher-order functions
- Closures
- String data structure
- Array data structure
- Hash data structure

## Usage

```bash
go run main.go
```
